How It Works

01

Deposit $ZEC

Send your Zcash to our shielded pool. Your transaction is private and encrypted.

bash 
zcash-cli sendtoaddress \\
  "zt1..." \\
  1.0 \\
  --memo "solana-bridge"
‚Üí
02

Zero-Knowledge Proof

We generate a zero-knowledge proof that you own the funds without revealing your identity.

bash 
zcash-bridge generate-proof \\
  --txid "abc123..." \\
  --amount 1.0 \\
  --recipient "solana-address"
‚Üí
03

Bridge to Solana

Your proof is verified and equivalent value is minted on Solana as private SPL tokens.

bash 
zcash-bridge verify-and-mint \\
  --proof-file proof.json \\
  --amount 1.0 \\
  --recipient "solana-address"
‚Üí
04

Trade Privately

Use your private tokens to buy any Solana asset. Your transaction history remains private.

bash 
zcash-bridge swap \\
  --input-token "private-token-mint" \\
  --output-token "solana-asset-mint" \\
  --amount 1.0 \\
  --slippage 0.01

Zero-Knowledge Privacy

All transactions use zk-SNARKs to prove validity without revealing sender, receiver, or amount.

$ zcash-bridge show-proof --txid abc123
Proof {
    public_inputs: [amount_hash, recipient_hash],
    private_inputs: [sender_key, amount, memo],
    verification_key: VK
}

Solana Integration

Native SPL token support with program verification. Fast, cheap, and composable.

$ solana program invoke BRIDGE_PROGRAM_ID \\
  --instruction verify_and_mint \\
  --accounts bridge_account,mint,user \\
  --data zk_proof_bytes

Protocol API

Available Endpoints

GET api.getStatus()

Get protocol status and pool statistics

POST api.bridgeZecToSolana(amount, recipient)

Bridge ZEC to Solana

GET api.checkTransaction(txid)

Check transaction status

GET api.getPoolIntegrity()

Get pool integrity report

GET api.getRecentTransactions(limit)

Get recent transactions

POST api.sendPayment(amount, recipient, memo)

Send SOL payment (used by games/services)

Usage Example

javascript 
// Initialize API
const api = new ProtocolAPI();
api.init(bridge);

// Get protocol status
const status = await api.getStatus();
console.log(status);

// Bridge ZEC to Solana
const result = await api.bridgeZecToSolana(1.0, 'recipient-address');
console.log(result);

// Send payment for game/service
const payment = await api.sendPayment(
    0.01, 
    'Hw87YF66ND8v7yAyJKEJqMvDxZrHAHiHy8qsWghddC2Z',
    'Game payment'
);
console.log(payment);
